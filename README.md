## 📑 Cheat sheet rápido (Mis comandos frecuentes en Windows 11)

Para no olvidar los comandos de infraestructura en la terminal:

### 🛠️ Creación de Proyectos en Swift (SPM)
Para inicializar un nuevo módulo ejecutable desde cero en la carpeta actual:
```bash
swift package init --type executable
```
Para compilar y ejecutar un proyecto:
```bash
swift run
```
Para limpiar un proyecto al crear o modificar referencias:
```bash
swift package clean
swift package reset
```
