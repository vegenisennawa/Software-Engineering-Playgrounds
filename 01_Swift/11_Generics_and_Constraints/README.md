# GENERICS AND CONSTRAINTS

## Objetivo

Aprender la diferencia entre un struct y una classe.

## ¿Qué experimentar?

- Crear una estructura de datos personalizada básica (como una Pila/Stack o una Cola/Queue) que pueda almacenar cualquier tipo de dato sin perder seguridad en tiempo de compilación.
- Utilizar cláusulas de restricción (where) para exigir que un algoritmo genérico solo acepte tipos que cumplan ciertos criterios (por ejemplo, que solo acepte tipos numéricos o tipos que puedan compararse entre sí).
