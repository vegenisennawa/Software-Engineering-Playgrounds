# JSON CODABLE DTOS

## Objetivo

Aprender la diferencia entre un struct y una classe.

## ¿Qué experimentar?

- Diseñar modelos DTO y conformar el protocolo mágico Codable.
- Observar cómo el compilador sintetiza automáticamente los parsers de JSON en tiempo de compilación (cero costo de reflexión en tiempo de ejecución).
- Practicar el mapeo de nombres de campos distintos (ej. convertir el formato snake_case de una API externa hacia la convención camelCase de tus modelos en Swift).
